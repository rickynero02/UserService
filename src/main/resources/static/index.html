File Sharing

File Sharing is a data sharing cloud platform with a social approach, developed by Alessio Acquas (UI), Raffaele Esposito & Riccardo Nisidi (Cloud & Server).

Development

In order to develop the best possible platform, we made use of some frameworks including AbstractCSS, developed internally, and Vue.js. Thanks to the developers

Design

For the design we used Material Design, released by Google in 2014, we also introduced ColorUI, in order to improve the user experience even more.
